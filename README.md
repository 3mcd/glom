# glom

[![codecov](https://codecov.io/github/3mcd/glom/graph/badge.svg?token=XCE84MWD6J)](https://codecov.io/github/3mcd/glom)

Glom is an Entity-Component System written in TypeScript. It aims to be a fast, networkable, and testable foundation for games with dynamic behavior.

[Read the docs](https://3mcd.github.io/glom/).
